# Daily Driver Workflow

## Launch Apps
- Terminal: SUPER + Enter
- Browser: SUPER + B
- File Manager: SUPER + E
- Launcher: SUPER + Space

## Screenshots
- Copy region: SUPER + P
- Save region: SUPER + S

## Clipboard

cliphist list
cliphist decode <id>

Mako handles notifications.
