# configs/

This directory contains all configuration files for Anthonyware OS.

Each subfolder corresponds to a specific application or subsystem:
- hypr/ — Hyprland window manager
- waybar/ — Waybar status bar
- kitty/ — Kitty terminal
- mako/ — Mako notifications
- eww/ — Eww widgets
- wofi/ — Wofi launcher
- systemd/ — User-level systemd services

These configs are loaded automatically when Hyprland starts.
